# E-postmal Problemer - Notater for Senere

## 🚨 Hovedproblemer

### 1. Logo Base64 Embedding
- **Problem**: Logo vises ikke i e-poster
- **Årsak**: Base64-string er for lang eller feil formatert
- **Løsning**: Krever dedikert dag for å fikse
- **Status**: Pending

### 2. Azure Exchange Blokkering
- **Problem**: Gmail/ProtonMail blokkeres med `550 5.7.708`
- **Årsak**: Microsoft Exchange aggressive spam-filtering
- **Løsning**: Mail Flow Rules i Microsoft 365 Admin Center
- **Status**: Pending

### 3. E-postmal Design
- **Problem**: Maler ser ikke profesjonelle ut
- **Årsak**: Kompleks HTML/CSS for e-postklienter
- **Løsning**: Bruke Stripo eller lignende tjeneste
- **Status**: Pending
